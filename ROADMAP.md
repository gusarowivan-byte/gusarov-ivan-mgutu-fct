# Дорожная карта проекта V-Avatar

В данном документе описаны ключевые этапы развития проекта: от учебного MVP до полноценной системы моделирования корпоративного обучения.

## Этап 1: Исследование и MVP (Выполнено ✅)
- [x] Анализ научного задела ФЦТ МГУТУ (ВКР 2024-2025 гг.).
- [x] Реализация базового модуля `legacy_methods.py`.
- [x] Настройка CI/CD (GitHub Actions) и инфраструктуры.
- [x] Регистрация команды на Leader-ID и публикация релиза v0.1.0.

## Этап 2: Расширение функционала (Краткосрочные цели — 1-2 месяца)
- [ ] **Интеграция с VR-оборудованием:** Тестирование работы аватаров с популярными VR-шлемами.
- [ ] **Улучшение UI/UX:** Разработка графического интерфейса для настройки параметров обучения без правки кода.
- [ ] **Расширение базы методов:** Внедрение еще 2-3 алгоритмов из смежных научных работ ФЦТ.

## Этап 3: Масштабирование и пилотирование (Среднесрочные цели — 6 месяцев)
- [ ] **Пилотный запуск в МГУТУ:** Проведение тестового занятия для студентов с использованием V-Avatar.
- [ ] **Система аналитики:** Разработка модуля автоматического сбора отчетов об успеваемости обучаемых.
- [ ] **Сообщество:** Привлечение более 10 активных контрибьюторов через open-source платформы.

## Этап 4: Продукт (Долгосрочные цели)
- [ ] Выход на полноценное коммерческое использование в корпоративном секторе.
- [ ] Интеграция с популярными LMS-системами.
