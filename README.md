Проект от ФЦТ МГУТУ  
Разрабатывается студентами Факультета цифровых технологий МГУТУ им. К.Г. Разумовского в рамках подготовки к акселераторам Leader-ID. Презентация проекта: https://pt.2035.university/project/v-avatar-tehnologia-personalizirovannyh-virtualnyh-avatarov-dla-obucenia-i-cifrovyh-servisov

Научная основа
Проект продолжает исследования, проведённые в МГУТУ:
ВКР: «Применение AR/VR-технологий в обучении сотрудников» направление подготовки 09.03.03 Прикладная информатика (2025)
Ключевой метод: Методы создания интерактивного контента.
Применение интерактивных элементов: взаимодействие с 3D-экспонатами, мультимедийные материалы, симуляции, тестирование знаний.

Чек-лист первого контрибьюта: 
1.Сделайте форк репозитория (кнопка Fork в правом верхнем углу).
2.Клонируйте свой форк локально: 
  git clone https://github.com/gusarowivan-byte/gusarov-ivan-mgutu-fct.git
3.Перейдите в папку проекта:
  cd gusarov-ivan-mgutu-fct
4.Установите зависимости:
  pip install -r requirements.txt
5.Запустите тесты:
  python -m pytest tests/
6.Внесите правки и отправьте PR (Pull Request).
